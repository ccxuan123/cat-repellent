# cat-repellent
small arduino project to scare cat away

## Component list
- 1 x Arduino Uno
- 1 x Ultrasonic sensor
- 1 x LED
- 1x 1k ohm resistor (check [here](https://eepower.com/resistor-guide/resistor-applications/resistor-for-led/#) to calculate the most suitable resistor for your LED)
- 1 x Piezo buzzer
- 1 x SG90 Micro Servo
- Some wires
- 1 x Power bank (I use it as power supply)
